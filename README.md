## Build Setup:

``` bash
# Download repository:
git clone https://github.com/petrovelykyi/gulp-template.git

# Go to the app:
cd gulp-template

# Install dependencies:
npm install

# Server with hot reload at http://localhost:3000/
gulp

# Output will be at build/ folder
gulp build
